# Web Research

Use this skill when asked to research a topic, find information online, or verify facts.

## Procedure

1. **Plan the search**: Identify 2–3 specific search queries that will yield high-quality results.
2. **Execute searches**: Use the browser tool to navigate to a search engine or directly to authoritative sources.
   ```
   navigate("https://www.google.com/search?q=<your+query>")
   ```
3. **Extract information**: Take a screenshot or read the page's accessible text to gather content.
4. **Cross-reference**: Verify important facts with at least one additional source.
5. **Summarise**: Write a concise, factual summary with sources cited as URLs.
6. **Save key facts**: Use `memory_save` to record any insights worth retaining long-term.

## Tips

- Prefer primary sources (official docs, papers, gov sites) over aggregators.
- For technical topics, check GitHub, official documentation, and Stack Overflow.
- If a page is paywalled, try an archive URL: `https://archive.ph/<url>`
- Always report uncertainty clearly. Never hallucinate sources.
